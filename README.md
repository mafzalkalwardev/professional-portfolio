# Muhammad Afzal Kalwar - Professional Portfolio

Animated multi-page developer portfolio website.

**Live site:** [mafzalkalwardev.github.io](https://mafzalkalwardev.github.io)

## Pages

| Page | Description |
|------|-------------|
| [index.html](index.html) | Hero, typing animation, stats, featured projects |
| [about.html](about.html) | Bio, services, experience timeline |
| [projects.html](projects.html) | Full project grid with category filters |
| [skills.html](skills.html) | Animated skill bars & tech stack |
| [contact.html](contact.html) | Contact form & social links |

## Features

- Fully animated UI (scroll reveal, counters, skill bars, page transitions)
- Responsive dark theme
- No build step - pure HTML, CSS, JavaScript
- GitHub Pages ready

## Local preview

Open `index.html` in a browser, or:

```bash
python -m http.server 8080
```

Then visit `http://localhost:8080`
